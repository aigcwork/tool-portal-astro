# 🛠️ 工具门户网站 (Tool Portal)

一个现代化的工具聚合网站，基于Astro框架构建，支持工具管理、分类浏览和全文搜索功能。

## ✨ 功能特性

### 🎯 核心功能
- **工具展示**: 美观的工具卡片展示，包含图标、描述和分类
- **智能搜索**: 实时搜索和筛选功能
- **分类管理**: 按类别浏览工具
- **响应式设计**: 完美适配移动端和桌面端

### 🔧 管理功能
- **工具管理**: 完整的CRUD操作界面
- **数据导入/导出**: 支持JSON格式的数据交换
- **本地存储**: 数据保存在浏览器本地存储中

### 📚 信息页面
- **功能说明**: 详细介绍网站功能和优势
- **使用指南**: 完整的操作指南
- **常见问题**: FAQ解答
- **最佳实践**: 使用建议和技巧
- **案例展示**: 实际应用案例
- **新闻资讯**: 最新动态和更新
- **集成扩展**: API和集成说明
- **安全隐私**: 隐私政策和安全措施

### 🎨 用户体验
- **深色模式**: 支持明暗主题切换
- **无障碍访问**: 完整的ARIA支持
- **性能优化**: 快速加载和流畅交互
- **SEO优化**: 搜索引擎友好的结构

## 🚀 快速开始

### 本地开发
```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

### 部署到Cloudflare Pages
```bash
# 安装wrangler
npm install -g wrangler

# 部署
npm run deploy
```

## 📁 项目结构

```
├── src/
│   ├── components/          # Astro组件
│   ├── layouts/            # 页面布局
│   ├── pages/              # 页面路由
│   ├── data/               # 数据文件
│   ├── utils/              # 工具函数
│   └── types/              # TypeScript类型定义
├── public/                 # 静态资源
├── dist/                   # 构建输出
└── wrangler.toml          # Cloudflare配置
```

## 🛠️ 技术栈

- **框架**: Astro
- **样式**: Tailwind CSS
- **语言**: TypeScript
- **部署**: Cloudflare Pages
- **图标**: Lucide React
- **状态管理**: 本地存储

## 📋 环境要求

- Node.js 18+
- npm 或 pnpm

## 🔧 配置说明

### 工具管理
访问 `/admin/tools` 路径可以管理工具数据，包括：
- 添加新工具
- 编辑现有工具
- 删除工具
- 导入/导出数据
- 搜索和筛选

### 主题切换
网站支持自动主题检测和手动切换，基于用户系统偏好。

### 响应式断点
- 移动端: < 768px
- 平板端: 768px - 1024px
- 桌面端: > 1024px

## 📝 开发指南

### 添加新页面
1. 在 `src/pages/` 目录创建 `.astro` 文件
2. 使用现有的布局组件
3. 遵循项目样式规范

### 修改工具数据
- 静态数据: 编辑 `src/data/tools.ts`
- 动态管理: 使用管理界面

### 样式规范
- 使用Tailwind CSS工具类
- 遵循移动端优先原则
- 保持组件模块化
